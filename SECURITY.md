# Security Policy

Please do not report security vulnerabilities in a public issue.

Email [simplecityadmin@gmail.com](mailto:simplecityadmin@gmail.com) with:

- a concise description of the issue;
- the affected page, endpoint, workflow, or file;
- steps to reproduce it safely; and
- the potential impact.

Do not access, modify, download, or delete data that does not belong to you. Do not include credentials, personal information, or other sensitive data in the report.

We will acknowledge the report as soon as we can and coordinate disclosure after a fix is available.
